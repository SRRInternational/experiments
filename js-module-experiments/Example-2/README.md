# My Modules

This project demonstrates how to create and use modules in UMD, AMD, CJS, and ESM formats.

## How to Run

1. **CJS:**

   ```sh
   npm run start
   ```

2. **ESM:**

    ```
    npm run start-esm
    ```

3. **AMD:**

    ```
    npm run start-amd
    ```

4. **UMD:**

    ```
    npm run start-umd
    ```
